# goodEvening Function

## Overview
The new `goodEvening` function expands our greeting library's capabilities by adding evening-specific greeting messages. This complements the existing morning and afternoon functions to provide a complete set of time-based greetings.

## Features

- **Evening-specific greetings**: Tailored messages for evening interactions
- **Personalization**: Customize greetings with an optional name parameter
- **Time-aware**: Automatically detects the appropriate greeting based on time of day
- **Seamless integration**: Works smoothly with all existing greeting-lib functionality

## Usage

```javascript
// Basic usage
const greeting = goodEvening(); // Returns "Good evening!"

// With name parameter
const personalGreeting = goodEvening("Alex"); // Returns "Good evening, Alex!"
```

## Integration with Existing Functions

The `goodEvening` function completes our suite of time-based greetings, working alongside the existing morning and afternoon functions to provide appropriate greetings throughout the day.
